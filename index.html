How?

Devices?

Tily is a fully fluid system meaning that the tiles will fit wherever you put them. But the default configuration ships with transforming rows and tiles. Meaning that you can define device specific row configurations and tiles that show as normal, wide or double depending on device

Browser support?

Tily is built to support modern and semi-modern browsers that support the box-sizing: border-box; CSS attribute. This means that IE8 and up is supported along with most other browsers.
